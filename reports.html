Cybersecurity Penetration Testing Report

1. Torilo Academy Penetration Testing

Scope

Tested toriloacademy.com including subdomains, web servers, and associated services. Testing phases: Information Gathering, Scanning, Vulnerability Assessment, Exploitation, Post-Exploitation.

Methodology

  • WHOIS Lookup
  • DNS & Subdomain Enumeration
  • Port Scanning (Nmap, Masscan)
  • Web Vulnerability Scanning (Nikto, WPScan, Dirb)
  • WAF Detection (WAFW00F)
  • OSINT Information Gathering (Spiderfoot)
  • Packet Capture & Analysis (Wireshark)

Findings

Conclusion: Minimal vulnerabilities found. Recommended: regular vulnerability scanning and patching outdated web components.

2. Bet9ja Penetration Testing

Scope

Authorized penetration test on bet9ja.com covering domain, subdomains, web servers, and databases. Focus: information gathering, vulnerability scanning, ethical exploitation.

Methodology

  • WHOIS Lookup
  • DNS & Subdomain Enumeration
  • Port Scanning (Nmap, Masscan)
  • Web Vulnerability Scanning (Nikto, WPScan, Dirb)
  • Database Vulnerability Testing (SQLMap)
  • Post-Exploitation Commands
  • Stealth Testing (MAC Spoofing, VPN, ProxyChains)

Findings

Conclusion: Database vulnerabilities were identified and ethically exploited. Recommendations: database hardening, secure authentication, and regular penetration testing.
